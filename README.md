# 印刷用 迷路ジェネレーター (Web版)

このツールは、印刷（A4サイズ）に最適な迷路をランダムに生成し、PDFとして保存できるツールです。

## ✨ 特徴
- **ランダム生成**: 実行するたびに異なる迷路が作られます。
- **印刷最適化**: 「PDF/印刷」ボタンを押すと、操作メニューを隠して迷路だけをA4サイズで出力します。
- **ブラウザで完結**: 面倒なインストールなしで、スマホやPCからすぐに使えます。

## 🚀 使い方
1. 幅と高さ（奇数を推奨）を入力して「生成」ボタンを押します。
2. 「PDF / 印刷」ボタンを押し、プリンタ設定で「PDFに保存」を選んでください。

## 🛠 技術
- HTML5 / Canvas API
- CSS Media Queries (Print)
